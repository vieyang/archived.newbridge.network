# Icon

Easily to add icons.

Using [Bootstrap Icons](https://icons.getbootstrap.com)

```tpl
{{</* icon icon=[iconname] class=[classname] color=[color] */>}}
```

## Example

```tpl
{{</* icon icon="cloud" */>}}
{{</* icon icon="cloud" color="red" */>}}
```
{{< icon icon="cloud" >}}
{{< icon icon="cloud" color="red" >}}
